# Ensign
Sublime Text Color Scheme

Current Version : **3.2**

An enhanced Sublime Text's dark color scheme. Color sheme for highliting following languages :

+ HTML
+ CSS
+ JS
+ PHP
+ Json (20 level deep arrays)
+ XML
+ Python
+ Nunjucks
+ Twig
+ Jade
+ Coffee
+ Plist
+ Diff
+ Markdown
+ C
+ Ruby
+ YAML
+ Find &amp; Replace
+ GitGutter
+ SublimeLinter
+ and Wiki based text.
